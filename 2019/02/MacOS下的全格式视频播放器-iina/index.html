MacOS下的全格式视频播放器-iina

发表于 2019-02-12
本文字数： 225 | 阅读时长 ≈ 1 分钟

刚接触Mac时，一直找不到好用的本地视频播放器，尝试过MplayerX、VLC，也付费过射手影音，最终在Github上找到了iina，该播放器开源免费，完全符合日常看视频的需求。整体界面整洁清爽，设计风格也完全贴合MacOS，符合大众审美。


安装方式有2种：

  1. Github下载最新版本，下载后直接安装。
  2. Mac上需要先安装Homebrew，同时安装iina需用到cask插件库，然后打开终端并运行：
    brew cask install iina
